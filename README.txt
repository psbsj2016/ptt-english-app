📁 ESTRUTURA:
- frontend/  → Arquivos do app (HTML, PWA)
- backend/   → Servidor Node.js + API

🚀 COMO INSTALAR:

1. Instale Node.js (nodejs.org)

2. Abra o terminal na pasta backend:
   cd backend

3. Instale as dependências:
   npm install

4. Inicie o servidor:
   npm start

5. Acesse no navegador:
   http://localhost:3000

📱 TESTAR PWA:
- Acesse pelo Chrome
- Clique em "Instalar" no banner
- Ou: Menu → "Instalar aplicativo"

🔧 ARQUIVOS EDITÁVEIS NO BLOCO DE NOTAS:
- package.json  → Dependências
- .env          → Configurações
- server.js     → Servidor
- database.js   → Banco de dados
- routes.js     → API endpoints

💾 BANCO DE DADOS:
- database.sqlite (criado automaticamente)
- Pode ser aberto com DB Browser for SQLite

🌐 DEPLOY GRATUITO:
